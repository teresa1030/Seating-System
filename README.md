# 簡易版員工座位系統
後端使用Java SpringBoot建立、前端使用Vue.js開發一個簡易版的員工座位系統

## 功能介紹
 * 新增座位
 * 新增員工
 * 更改員工座位

## 開發環境
後端
  * Spring Boot
  * MySQL

前端
  * Vue.js
